# Пошаговая инструкция деплоя на Vercel

## Визуальное руководство по интерфейсу Vercel

### Шаг 1: Вход в Vercel

1. Откройте https://vercel.com
2. Нажмите **"Sign Up"** (если нет аккаунта) или **"Log In"**
3. Выберите **"Continue with GitHub"**
4. Разрешите доступ к репозиториям

### Шаг 2: Создание проекта

После входа вы увидите Dashboard Vercel:

1. В правом верхнем углу найдите кнопку **"Add New..."** (или просто **"New Project"**)
2. Нажмите на неё
3. Выберите **"Project"** из выпадающего меню

### Шаг 3: Импорт репозитория

Вы увидите страницу "Import Git Repository":

1. В списке репозиториев найдите **`knowflow`**
2. Если репозитория нет в списке:
   - Нажмите **"Adjust GitHub App Permissions"**
   - Выберите репозиторий `knowflow`
   - Нажмите **"Install"**
3. Рядом с репозиторием `knowflow` нажмите кнопку **"Import"**

### Шаг 4: Настройка проекта

После импорта откроется страница "Configure Project":

**Что вы увидите:**

```
┌─────────────────────────────────────┐
│  Configure knowflow                 │
├─────────────────────────────────────┤
│                                     │
│  Framework Preset: Next.js ✓        │
│  Root Directory: ./                 │
│  Build Command: npm run build       │
│  Output Directory: .next            │
│  Install Command: npm install       │
│                                     │
│  [Environment Variables]            │
│  ┌─────────────────────────────┐   │
│  │ + Add Variable              │   │
│  └─────────────────────────────┘   │
│                                     │
│  [внизу страницы]                   │
│  ┌─────────────────────────────┐   │
│  │      [Deploy]               │   │ ← ВОТ ОНА!
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

**Важно:** Не нажимайте "Deploy" сразу! Сначала добавьте переменные окружения.

### Шаг 5: Добавление переменных окружения

1. Найдите секцию **"Environment Variables"** (примерно посередине страницы)
2. Нажмите кнопку **"+ Add"** или **"Add Variable"**
3. Появится форма:

```
┌─────────────────────────────────────┐
│  Name: [________________]           │
│  Value: [________________]           │
│  Environment:                       │
│  ☑ Production                       │
│  ☑ Preview                          │
│  ☑ Development                      │
│  [Cancel]  [Add]                    │
└─────────────────────────────────────┘
```

4. Добавьте переменные по очереди:

   **Переменная 1:**
   - Name: `NEXT_PUBLIC_SUPABASE_URL`
   - Value: ваш URL из Supabase (например: `https://xxx.supabase.co`)
   - Environment: отметьте все три (Production, Preview, Development)
   - Нажмите **"Add"**

   **Переменная 2:**
   - Name: `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - Value: ваш anon key из Supabase
   - Environment: все три
   - Нажмите **"Add"**

   **Переменная 3:**
   - Name: `TELEGRAM_BOT_TOKEN`
   - Value: `8270666542:AAE2M6MUIEM2KUW1GpWUKG52aszkj70x5fA`
   - Environment: все три
   - Нажмите **"Add"**

   **Переменная 4:**
   - Name: `TELEGRAM_CRON_SECRET`
   - Value: любая случайная строка (например: `my-secret-key-123`)
   - Environment: все три
   - Нажмите **"Add"**

5. После добавления всех переменных прокрутите страницу вниз

### Шаг 6: Нажатие кнопки Deploy

1. Прокрутите страницу до самого низа
2. Вы увидите большую синюю кнопку **"Deploy"**
3. Нажмите на неё

**Где находится кнопка:**
- В самом низу страницы настройки
- Большая синяя кнопка
- Текст: "Deploy"

### Шаг 7: Ожидание деплоя

После нажатия "Deploy":

1. Вы будете перенаправлены на страницу деплоя
2. Увидите процесс сборки в реальном времени:
   ```
   [●] Cloning repository...
   [●] Installing dependencies...
   [●] Building...
   [●] Deploying...
   ```

3. Дождитесь завершения (обычно 2-3 минуты)
4. Когда увидите ✅ "Congratulations!" - деплой завершен

### Шаг 8: Получение URL

После успешного деплоя:

1. Вверху страницы вы увидите URL:
   - `https://knowflow-xxx.vercel.app` или
   - `https://your-custom-domain.vercel.app`

2. Скопируйте этот URL
3. Откройте его в браузере - ваше приложение должно работать!

## Если не видите кнопку Deploy

1. **Проверьте, что вы на странице настройки проекта**
   - URL должен быть: `vercel.com/new/...` или `vercel.com/.../settings`

2. **Прокрутите страницу вниз**
   - Кнопка всегда в самом низу

3. **Проверьте, что репозиторий импортирован**
   - Должна быть видна информация о проекте

4. **Попробуйте обновить страницу** (F5)

## Альтернативный способ

Если не можете найти кнопку "Deploy":

1. После импорта репозитория Vercel может автоматически начать деплой
2. Проверьте вкладку **"Deployments"** в Dashboard
3. Если деплой уже начался - просто дождитесь завершения

## После деплоя

1. Скопируйте URL приложения
2. Установите Telegram webhook (см. QUICK_DEPLOY.md, Шаг 6)
3. Проверьте работу приложения в браузере
