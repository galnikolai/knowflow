# Настройка AI для генерации флеш-карточек

## Быстрый старт с OpenAI (Рекомендуется)

### 1. Получите API ключ OpenAI

1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в раздел API Keys
3. Создайте новый ключ

### 2. Добавьте ключ в переменные окружения

Создайте файл `.env.local` в корне проекта (если его еще нет):

```bash
OPENAI_API_KEY=sk-your-api-key-here
```

### 3. Перезапустите сервер разработки

```bash
npm run dev
```

## Альтернативные варианты

### Anthropic Claude

1. Получите API ключ на https://console.anthropic.com
2. Добавьте в `.env.local`:
```bash
ANTHROPIC_API_KEY=sk-ant-your-api-key-here
```

### Ollama (Локально)

1. Установите Ollama: https://ollama.ai
2. Скачайте модель:
```bash
ollama pull llama3.2
```
3. Убедитесь, что Ollama запущен (по умолчанию на http://localhost:11434)
4. В коде используйте `provider: "ollama"`

## Использование

После настройки, при нажатии на кнопку "Изучить" → "Создать" в диалоге, система автоматически:

1. Извлечет текст из заметки
2. Отправит его в выбранную AI модель
3. Сгенерирует 8 флеш-карточек
4. Сохранит их в базу данных

## Стоимость (примерная)

- **OpenAI GPT-4o-mini**: ~$0.15 за 1M входных токенов
- Для заметки на 1000 слов (~1300 токенов): ~$0.0002
- **Anthropic Claude**: ~$3 за 1M входных токенов
- **Ollama**: Бесплатно (локально)

## Безопасность

⚠️ **Важно**: API ключи хранятся только на сервере в `.env.local` и никогда не отправляются на клиент.
