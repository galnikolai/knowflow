# Рекомендации по выбору AI модели для генерации флеш-карточек

## Варианты моделей

### 1. OpenAI GPT-4o / GPT-4o-mini (Рекомендуется для начала)

**Плюсы:**
- ✅ Отличное качество генерации
- ✅ Быстрые ответы
- ✅ Простая интеграция
- ✅ Хорошо понимает контекст и структурирует данные

**Минусы:**
- ❌ Платно (~$0.15-0.60 за 1M токенов для GPT-4o-mini)
- ❌ Требует API ключ
- ❌ Зависимость от интернета

**Стоимость:** ~$0.15-0.60 за 1M входных токенов (GPT-4o-mini)

**Установка:**
```bash
npm install openai
```

### 2. Anthropic Claude 3.5 Sonnet

**Плюсы:**
- ✅ Очень хорошее качество
- ✅ Длинный контекст (200K токенов)
- ✅ Хорошо структурирует данные

**Минусы:**
- ❌ Платно (~$3 за 1M входных токенов)
- ❌ Требует API ключ

**Установка:**
```bash
npm install @anthropic-ai/sdk
```

### 3. Ollama (Локальная модель)

**Плюсы:**
- ✅ Бесплатно
- ✅ Полная приватность (данные не уходят в облако)
- ✅ Работает офлайн
- ✅ Нет лимитов на использование

**Минусы:**
- ❌ Требует установки Ollama на компьютер
- ❌ Может быть медленнее облачных решений
- ❌ Требует больше ресурсов (RAM, GPU)

**Рекомендуемые модели Ollama:**
- `llama3.2` (3B) - быстрая, легкая
- `mistral` (7B) - хороший баланс
- `qwen2.5` (7B) - хорошее качество

**Установка:**
1. Установить Ollama: https://ollama.ai
2. Скачать модель: `ollama pull llama3.2`
3. Использовать через HTTP API

### 4. Hugging Face Inference API

**Плюсы:**
- ✅ Бесплатные модели
- ✅ Простое API
- ✅ Много моделей на выбор

**Минусы:**
- ❌ Может быть медленнее
- ❌ Лимиты на бесплатном тарифе
- ❌ Качество может быть ниже

## Рекомендация

**Для начала:** Используйте **OpenAI GPT-4o-mini** - это самый простой и быстрый способ получить хорошие результаты.

**Для приватности:** Добавьте поддержку **Ollama** как альтернативу для пользователей, которые хотят локальную обработку.

## Пример интеграции

См. файл `src/shared/api/ai-generator.ts` для примера реализации.

## Структура промпта для генерации флеш-карточек

```
Ты - помощник для создания флеш-карточек для запоминания информации.

На основе следующего текста создай набор флеш-карточек для эффективного запоминания.

Текст:
{content}

Требования:
1. Создай 5-10 флеш-карточек
2. Вопросы должны быть конкретными и проверять понимание
3. Ответы должны быть краткими, но информативными
4. Используй формат JSON:
{
  "cards": [
    {
      "question": "Вопрос",
      "answer": "Ответ"
    }
  ]
}
```
