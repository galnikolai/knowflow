# Быстрый старт с Ollama

## Шаг 1: Установка (5 минут)

### macOS
```bash
brew install ollama
```

### Windows/Linux
Скачайте с https://ollama.ai/download

## Шаг 2: Запуск Ollama

Ollama должен запуститься автоматически. Если нет:
```bash
ollama serve
```

## Шаг 3: Скачивание модели (2-5 минут)

```bash
ollama pull llama3.2
```

Это займет несколько минут в зависимости от скорости интернета (~2GB).

## Шаг 4: Проверка

```bash
ollama run llama3.2 "Привет"
```

Если видите ответ — всё готово! ✅

## Шаг 5: Использование в приложении

1. Откройте заметку
2. Нажмите на троеточие (⋮) → "Изучить"
3. Выберите "Создание флеш-карточек"
4. Выберите **"Ollama (локально, бесплатно)"**
5. Выберите модель (Llama 3.2 рекомендуется)
6. Нажмите "Создать"

Готово! Флеш-карточки будут созданы локально на вашем компьютере.

## Решение проблем

### "Ошибка подключения к Ollama"
- Убедитесь, что `ollama serve` запущен
- Проверьте: `curl http://localhost:11434/api/tags`

### "Модель не найдена"
```bash
ollama pull llama3.2
```

### Медленная генерация
- Это нормально для CPU (10-30 секунд)
- Используйте более легкую модель (llama3.2)
- GPU значительно ускорит работу (автоматически используется, если доступен)

## Преимущества

✅ **Бесплатно** — никаких лимитов  
✅ **Приватно** — данные не покидают ваш компьютер  
✅ **Офлайн** — работает без интернета
